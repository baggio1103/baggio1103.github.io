Programming Basics with Kotlin

This course is intended for beginners or students with little programming experience who are willing to learn and advance their skills.

Module 1: Introduction to Kotlin programming language

  • Overview of Kotlin language features.
  • Advantages of using Kotlin compared to Java.
  • Setting Up Environment: Installing JDK and Kotlin compiler, Configuring IDE (IntelliJ IDEA recommended).

Module 2: Kotlin Basics

  • Variables and Data Types: Declaring variables and constants, Basic data types (Int, Double, Boolean, String).
  • Operators and Expressions: Arithmetic, comparison, and logical operators, Expressions and statements.

Module 3: Control Flow In Kotlin

  • Conditional Statements: if, else-if, else expressions, when expression (Kotlin's replacement for switch).
  • Loops: for loop, while loop, Using ranges and step in loops.

Module 4: Functions & Lambdas

  • Defining functions with parameters and return types.
  • Function overloading and default arguments.
  • Higher-Order Functions and Lambdas: Passing functions as arguments, Declaring and using lambda expressions.

Module 5: Collections & Data Structures

  • Arrays and Lists: Declaring arrays and lists, Accessing and modifying elements.
  • Maps and Sets: Creating and using maps and sets, Performing common operations.

Module 6: Object Oriented Programming in Kotlin

  • Classes and Objects: Defining classes and creating objects, Constructors and properties.
  • Inheritance and Polymorphism: Extending classes and overriding methods, Using interfaces and abstract classes.

Module 7: Error Handling & Exception Handling

  • Exception Handling: Using try-catch blocks, Throwing exceptions.
  • Error Handling with Kotlin: Handling nullability with nullable types and the Elvis operator, Safe calls and the !! operator.

Module 8: Kotlin Applications Development

  • Building Console Applications: Developing simple command-line programs, Handling user input and displaying output.

Module 9: Introduction to Android Development with Kotlin

  • Overview of Kotlin for Android development.
  • Building a basic Android app using Kotlin.

Bonus Section

Module 10: Building Our First Http Server

Outro

By the end of this course, you will gain solid and fundamental knowledge of basic programming and Kotlin language. You will be able to:

  • Write your own programs
  • Develop console applications
  • Feel confident with OOP (classes, objects, interfaces, etc)
  • Work with collections (lists, set and maps)
  • Analyze complexity of an algorithm

Last but not least, this course lets you dive into the world of programming and boost your career or enthusiasm towards computer science i.e. programming.
